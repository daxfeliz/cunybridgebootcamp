        <h1> General Information</h1>
          <p> The main purpose of the group is to: </p>
              <ul>
                  <li> Show tools and applications that are useful for research</li>
                  <li> Learn how to apply such tools to projects, especially using <em><b>Python</b></em></li>
                  <li> Teach good coding practices</li>
                  <li> Create a space where one can ask questions and easily get help from students, post-docs, and faculty</li>
                  <li> Improve your coding skills, and <i><b>have fun</b></i> while doing it!</li>
            </ul>
          <p> The <b>Github repository</b> for this class is: <a href="https://github.com/daxfeliz/cunybridgebootcamp" target="_blank">https://github.com/daxfeliz/cunybridgebootcamp</a></p>
          
        <h1>Slack Information</h1>
